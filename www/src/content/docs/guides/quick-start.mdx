---
title: Quick Start
description: Create a graph and run the first project queries.
---

Create an index for a project:

```bash
lexa index /path/to/project
cd /path/to/project
```

Run common queries from the project root:

```bash
lexa files
lexa text-search "handle_request"
lexa outline src/main.rs
lexa symbol-defs Engine
lexa read src/main.rs -L 1-80
```

`index` writes the graph to `.lexa/graph.lexa` by default. Commands run from
the project root read that graph automatically.

## Custom Graph Path

Use `--graph` when you want the index outside the project:

```bash
lexa --graph /tmp/project.graph.lexa index /path/to/project
lexa --graph /tmp/project.graph.lexa text-search "Parser"
```

## Safe Edit Example

```bash
lexa read src/main.rs --hash
lexa patch src/main.rs replace -L 12 --if-hash <hash> --content '    println!("updated");'
lexa create src/new_file.rs --content 'pub fn new_file() {}'
```
