---
title: Install
description: Build or install Lexa from source.
---

Lexa is a native Rust binary. Install it from the repository root with Cargo:

```bash
cargo install --path .
```

Or build a release binary without installing it:

```bash
cargo build --release
./target/release/lexa --help
```

Make sure Cargo's bin directory is available on your `PATH`:

```bash
export PATH="$HOME/.cargo/bin:$PATH"
```

## Verify

```bash
lexa --help
```

If the command prints the CLI help, the binary is ready to use.
