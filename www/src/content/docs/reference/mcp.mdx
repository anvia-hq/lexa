---
title: MCP Server
description: Expose Lexa as a local MCP server.
---

Expose the same graph-backed tools to an MCP client:

```bash
lexa mcp /path/to/project
```

Run MCP without loading or saving a graph:

```bash
lexa --no-graph mcp /path/to/project
```

Example MCP client config:

```json
{
  "mcpServers": {
    "lexa": {
      "command": "/path/to/lexa",
      "args": ["mcp", "/path/to/project"]
    }
  }
}
```

The MCP server uses the same local graph as the CLI, which keeps human terminal
workflows and agent workflows on one shared project view.
